# Mirage
Mirage is a desktop environment for [Cosmos](https://github.com/CosmosOS/Cosmos).

## System Requirements
* BIOS
* VM: VMware
* CPU: i386
* RAM: 80 MB
* Video: VMware SVGA-II
* Peripherals: PS/2
